## Description
GlobeTrotter is a travel planning web application that allows users to create trips,
add activities, manage itineraries, and estimate travel budgets.

## Features
- Login & Dashboard
- Create Trip
- Add Activities
- View Itinerary
- Budget Calculation
- SQLite Database

## Tech Stack
- Python
- Flask
- SQLite
- HTML

## How to Run
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```
